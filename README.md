## Reac-Docs


This is React docs a rich format text editor similar in the veins of Microsoft word or Google Docs.
This was a learning opportunity to build a fully functioning produciton ready application.

#### Tech Stack:
- NextJS
- Convex
- Clerk
- Tiptap
- Liveblocks
- Zustand

### Features:
- Font change, Size Change and format changes.
- collaborative editing.
- Notifications and tagging of multiple users.
- Organizations and personal accounts.
